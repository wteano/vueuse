# @vueuse/math

[![NPM version](https://img.shields.io/npm/v/@vueuse/math?color=a1b858)](https://www.npmjs.com/package/@vueuse/math)

> 这是 [VueUse](https://github.com/vueuse/vueuse) 的扩展，为组合式函数提供数学运算功能。

## 安装

```bash
npm i @vueuse/math @vueuse/core
```
